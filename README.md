## Python e DUCKDB

Este projeto é um estudo baseado no workshop do Luciano Vasconcelos, utilizando o Python, Docker e DuckDB, o objetivo aqui é o aprendizado do DuckDB para ETL para facilitar o tratamento e análise de dados.
